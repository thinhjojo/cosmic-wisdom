## 📦 Installation

### Option 1: Download & Install

```bash
# Clone this repo
git clone https://github.com/thinhjojo/cosmic-wisdom.git

# Copy to GoClaw skills directory
cp -r cosmic-wisdom ~/.goclaw/skills-store/

# Or for Docker
cp -r cosmic-wisdom /app/.goclaw/skills-store/
```

### Option 2: Publish to System

```bash
# Using GoClaw CLI
goclaw skills publish cosmic-wisdom

# Or manually copy
cp cosmic-wisdom/SKILL.md ~/.goclaw/skills-store/cosmic-wisdom/1/
```
